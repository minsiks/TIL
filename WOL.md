# WOL (Wake On Lan) 이란

- 네트워크에 연결된 컴퓨터의 전원을 켜기 위한 기능
- 컴퓨터가 꺼져 있는 상태에서 일정 신호를 통해 부팅

## 기대하는 필요성

- 세계 여행 중 네트워크만 연결이 되어 있다면 집에 있는 고사양 PC를 부팅, 원격 제어 가능
- 상대적으로 저 중량의 아이 패드를 갖고 다니며 사용, 상대적으로 고중량, 고사양의 PC를 사용 가능
- 세계 여행 중 맥북을 들고 다닐 예정이기에 때때로 익숙한 Window OS 환경으로 사용 가능
- 적어도 백업 기능 용도로도 사용 가능

## 사용 방법

## 1. 기기 및 사전 세팅

### 1-1 집 PC와 공유기 유선 연결

- 무선으로 PC 이용하고 있다면 WOL이 불가
- 꼭 공유기와 PC를 랜케이블 통해 유선 연결

### 1-2. 메인보드 및 네트워크 카드 WOL 지원 확인

- 요즘 출시되는 PC 대부분 이 기능을 지원

## 2. PC WOL 설정

### 2-1.  윈도우 장치관리자 실행

- 윈도우 작업표시줄 검색창 > 장치관리자 입력
- 네트워크 어댑터 > 속성 > 전원 관리 탭 > "이 장치를 사용하여 컴퓨터의 대기 모드를 종료할 수 있음", "매직 패킷에서만 이 컴퓨터의 대기 모드를 종료할 수 있음"을 모두 체크

### 2-2. PC UEFI / BIOS 설정 진입 or Wol 설정

### 2-3. PC 전원 옵션 설정

- 윈도우 8 이상은 ‘빠른 시작 켜기’가 활성화
- 이는 PC 종료시 LAN 카드에 전원을 공급하지 않아 WOL 사용 불가, 해제 필수
- 제어판 > 전원 옵션 > 전원 단추 작동 설정 >현재 사용할 수 없는 설정 변경 > 빠른 시작 켜기 해제

### 2-4. PC 물리적 주소 확인

- 맥 어드레스(물리적 주소)는 전 세계 단 하나뿐인 내 컴퓨터의 고유 주소
- 공유기와 유선 연결된 랜카드의 물리적 주소를 확인 할 것
- 명령프롬프트(cmd) 실행 > ipconifg / all > 물리적 주소 12자리 확인 (00-00-00-00-00-00)

## 3. 공유기 WOL 설정

### 3-1. WOL 깨우기 설정

<aside>
💡 공유기 관리자 페이지 접속

- IPTIME 관리자 페이지 접속

(주소창 : 192.168.0.1) > 고급설정 > 특수기능 > WOL 기능 메뉴

"현재 접속된 PC의 주소로 설정" 체크

- KT 공유기 로그인
http://172.30.1.254:8899/login/login
아이디 : ktuser
비밀번호 : homehub

</aside>

<로그인 화면 이미지 캡쳐>

- KT 공유기의 IP Address 확인
    - KT 공유기 로그인 → KT 공유기의 상태정보 → 시스템 정보 → 인터넷 연결정보 → IP주소:***.***.***.***(확인)
- [포트 포워딩] 설정하기
    - KT 공유기 로그인 → 장치설정 → 트래픽 관리 → [포트 포워딩 설정] 화면으로 진입
    - PC의 포트 포워딩 데이터 입력
        - 외부 포트 : 0 ~ 65,535 중 임의의 값 입력
        - 내부 IP 주소 : PC의 내부 IP 주소 입력
        - 내부 포트 : 0 ~ 65,535 중 임의의 값을 입력 하거나 외부포트와 동일한 값 입력
        - 프로토콜 : ALL
        - 설명 : 해당 포트 포워딩을 설명하는 임의의 값 입력
    
    ![캡처.PNG](https://prod-files-secure.s3.us-west-2.amazonaws.com/fa64aa7d-9054-4148-8d2a-f4be4bc9e749/b40fe9ca-5760-4764-83e7-32ef0f7c562a/%EC%BA%A1%EC%B2%98.png)
    

- 추가하면 하단의 포트 포워딩 리스트에 PC가 나타남

## 4. 원격 데스크톱 PC 전원 켜기

- WOL Control앱 아이패드에 설치
- 앱 실행 후 PC의 포트 포워딩 데이터를 저장
    - Host name / IP : 원격 KT 공유기의 외부 IP 주소 입력
    - Port : KT 공유기에서 포트 포워딩한 외부 포트 번호 입력
    - MAC address : 원격 PC의 맥어드레스 입력
- WOL Control 실행 → Quick Startup 화면 진입 → PC의 포트 포워딩 데이터 (입력) → [Send WOL] 터치 → PC 전원 켜짐 확인
